# MetalForge Roadmap

*Last updated: 2026-01-31*

## Vision
Become the #1 resource for metal drummer gear information, driving traffic through SEO and monetizing via affiliate partnerships.

**Goal:** Solid content base + basic monetization

### Week 1-2 ✅
- [x] 13 drummers with complete profiles
- [x] Video integration (drum cams)
- [x] SEO basics (sitemap, robots.txt, meta tags)
- [x] Google Analytics + Search Console

### Week 3-4
- [ ] Affiliate links integration (Thomann EU, Sweetwater US)
- [ ] Setup Cost Calculator (first monetization feature)
- [ ] 10 more drummers (target: 23 total)
- [ ] Individual gear pages (top 20 products)
