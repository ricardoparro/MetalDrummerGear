## Variables Reference

| Variable | Description |
|----------|-------------|
| `{DRUMMER_A}` | First drummer name |
| `{DRUMMER_B}` | Second drummer name |
| `{KIT_SUMMARY_A}` | Brief kit description (e.g., "TAMA Starclassic + Paiste") |
| `{KIT_SUMMARY_B}` | Brief kit description |
| `{BAND_NAME}` | Primary band |
| `{PERCENTAGE}` | Vote percentage |
| `{WINNER}` | Winning drummer |
| `{LOSER}` | Losing drummer |
| `{FAMOUS_TRACK}` | Iconic song featuring their drumming |
| `{METALFORGE_LINK}` | Link to drummer profile on site |
| `{BPM}` | Typical playing speed |
