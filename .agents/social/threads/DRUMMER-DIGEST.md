# 🥁 Drummer Digest Thread Series

Daily Twitter thread series featuring **"5 INSANE Facts About [DRUMMER]"** format.

## Thread Format

```
🥁 5 INSANE Facts About [DRUMMER] 🤘

1/ [Unique technique fact]
2/ [Gear quirk + price/detail]
3/ [Career milestone]
4/ [Influence/legacy]
5/ [Current status/trivia]

Full setup → metalforge.io/drummer/[slug]
Tag a drummer! 🤘

#MetalDrumming #DrumGear
```

## Posting Rules

- **Time:** 8 PM EST daily (peak metal community activity)
- **Thread format:** Each fact as a reply to the main tweet
- **Tag drummer:** In a REPLY (not main thread) — avoids spam feel
- **Reply to comments:** Within 1 hour of posting
- **Hashtags:** Max 2 (#MetalDrumming #DrumGear)
- **Pin:** Best-performing thread to profile
