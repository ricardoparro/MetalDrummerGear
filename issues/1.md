### Issue
The [Quotes](https://metalforge.io/quotes) page is not functioning as intended. Key features such as quotes display, search functionality, and filters are broken or missing entirely.

### Details
- **Broken Features:**
  - Quotes are not being displayed on the [Quotes page](https://metalforge.io/quotes).
  - Search functionality does not return any results when searching for quotes.
  - The filter for drummers is not working as expected.

### Steps to Reproduce
1. Navigate to [Quotes page](https://metalforge.io/quotes).
2. Attempt to search for a quote — no results appear.
3. Try applying a filter for drummers — the dropdown does not display a list of drummers.

### Expected Behavior
- Quotes should appear by default on the Quotes page.
- The search bar should allow users to find specific quotes.
- The filter should provide a dropdown of drummers, and selecting one should display the relevant quotes for that drummer.

### Additional Context
This page should be fixed so that users can:
- View quotes directly on the page.
- Search for quotes effectively.
- Properly filter and view quotes associated with specific drummers.
