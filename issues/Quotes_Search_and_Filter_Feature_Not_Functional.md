**Issue:**  The [Quotes page](https://metalforge.io/quotes) is not functioning as expected. Key features such as the search functionality and the filter for drummers do not work.

**Details:**
- The quote search does not return any quotes when text input is provided.
- The drummer list dropdown filter does not appear, preventing the filtering of quotes by drummers.
- Filtering should display quotes associated with the selected drummer, but this is not working.

**Steps to Reproduce:**
1. Navigate to the [Quotes page](https://metalforge.io/quotes).
2. Attempt to search for quotes using the search bar.
3. Observe that no quotes are displayed in the results.
4. Try applying the drummer filter and observe that the dropdown list does not appear.

**Expected Behavior:**
- Quotes should appear when searched by text in the search bar.
- The dropdown filter should list drummers, and selecting one should display quotes associated with that drummer.

**Improvement Suggestion:**
This feature needs better design and functionality. Tests should ensure that:
- Searching works and displays relevant results.
- Filtering by drummers shows accurate drummer-specific quotes.

**Impact:**
This bug prevents users from exploring and interacting with quotes effectively, reducing user experience and site functionality.
