## Bug: GA4 Counting CI Visits

### Root Cause
PR #421 (drumming techniques) accidentally changed playwright.config.cjs:
- **Before:** `?ci_test=1` appended to URL (worked)
- **After:** `X-CI-Test` HTTP header (doesn't work — GA4 can't see headers)

### Fix
Restore: `const TEST_BASE_URL = process.env.CI ? \`${BASE_URL}?ci_test=1\` : BASE_URL;`

### Prevention (#463)
1. Warning comment in code
2. E2E test that fails if ci_test missing
3. CI workflow check (grep)
4. Document in AGENTS.md as protected pattern
