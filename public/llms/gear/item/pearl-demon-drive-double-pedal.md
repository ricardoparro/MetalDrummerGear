# Pearl Demon Drive Double Pedal

**Category:** hardware | **Brand:** Pearl

---

## Overview

The Pearl Demon Drive is an advanced double bass pedal featuring the innovative NiNjA Bearing technology for ultra-smooth action. With interchangeable cams and click-lock spring tension, it offers unparalleled customization for demanding drummers.

## Specs

| Field | Value |
|-------|-------|
| Cam Type | Interchangeable (Direct/Standard) |
| Beater Type | Demon Beater (switchable) |
| Footboard | PowerShifter Long Board |
| Bearings | NiNjA Bearings |
