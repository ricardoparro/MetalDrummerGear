# Pearl Reference Series Drums

**Category:** drums | **Brand:** Pearl

## Overview

Pearl Reference drums feature the innovative Reference Shell construction with birch, maple, and mahogany plies. Each shell size is specifically designed with optimal wood combinations for that drum's role, delivering focused tone and maximum projection.
