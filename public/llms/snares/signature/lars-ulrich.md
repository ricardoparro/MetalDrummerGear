# What Snare Does Lars Ulrich Use? Tama LU1465 Lars Ulrich Signature

## Direct Answer

Lars Ulrich plays the Tama LU1465 Lars Ulrich Signature — a signature snare built for them, sized 14x6.5".

This is a namesake signature model — Tama built the LU1465 Lars Ulrich Signature specifically for Lars Ulrich of Metallica, rather than Lars Ulrich simply endorsing an off-the-shelf shell. That distinction is why this snare gets its own page: it's the drummer's own spec, not a generic stock configuration another drummer could also be playing.
