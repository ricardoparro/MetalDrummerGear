# MetalForge Dream Set Builder

> Build your ultimate metal drum kit by mixing and matching gear from legendary metal drummers.

**URL:** https://metalforge.io/tools/dream-set-builder
**Type:** Free interactive web tool

## What It Does

The MetalForge Dream Set Builder lets you construct a custom drum kit by selecting individual components — shells, snare, cymbals, pedals, and hardware — from the real gear used by 60+ professional metal drummers. Pick Lars Ulrich's Tama Starclassic shells, pair them with Danny Carey's Paiste Signature cymbals, add Gene Hoglan's Pearl double-bass pedals, and see the complete dream kit assembled on screen.

Each component selection shows which drummer made it famous, the exact model specs, and links to that drummer's full gear profile. The result is a shareable custom kit that blends influences from across the metal drumming world.

## Key Features

- **Shell selection:** Choose bass drums, toms, and floor toms from any roster drummer's kit
- **Snare picker:** Select signature and standard snare models used by the pros
- **Cymbal configurator:** Build hi-hat, crash, ride, and china combinations from real setups
- **Pedal & hardware:** Pick double-bass pedals and rack systems from the database
- **Shareable output:** Export your dream set as a named kit for referencing or sharing

## FAQ

**Q: What is the MetalForge Dream Set Builder?**
A: It's a free tool at metalforge.io/tools/dream-set-builder that lets you assemble a custom metal drum kit from real gear used by over 60 legendary professional metal drummers.

**Q: Can I mix gear from different drummers?**
A: Yes — that's the core concept. You can combine shells from one drummer, cymbals from another, and pedals from a third to build your ideal hybrid kit.

**Q: Is the gear data accurate to what the drummers actually play?**
A: All gear in the builder comes from MetalForge's verified drummer profiles, which document the actual endorsed and used setups for each artist.
