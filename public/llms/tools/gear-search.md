# MetalForge Gear Search Tool

> Find which metal drummers use specific gear by searching by brand, cymbal model, or drum kit.

**URL:** https://metalforge.io/tools/gear-search
**Type:** Free interactive web tool

## What It Does

MetalForge Gear Search lets you search across 60+ professional metal drummers' gear setups by brand (Pearl, Tama, Zildjian, Meinl, Sabian, Paiste, DW…), cymbal model, or drum kit name. Every search returns the full list of roster drummers who endorse or actively use that product, with links to their complete gear profiles.

The database covers drums, snares, cymbals, double-bass pedals, hardware, and drumsticks for each drummer — so a single search surfaces the complete picture of who plays what.

## Example Queries

- Search **"Pearl"** → all metal drummers using Pearl drums (Joey Jordison, Gene Hoglan, George Kollias, Ray Luzier, and 14 more)
- Search **"Zildjian A Custom"** → finds every drummer who plays that specific cymbal line
- Search **"Mapex"** → returns Chris Adler, Arin Ilejay, Jason Bittner
- Search **"Tama Iron Cobra"** → double-bass pedal users across the roster
- Search **"Meinl Byzance"** → Brann Dailor, Chris Adler, Matt Halpern, and others

## FAQ

**Q: How do I find which metal drummers use a specific brand?**
A: Use the MetalForge Gear Search at metalforge.io/tools/gear-search — type any brand name (e.g. "Sabian", "DW", "Vic Firth") to see every matching drummer with a link to their full gear profile.

**Q: Can I search by cymbal model, not just brand?**
A: Yes. The search covers specific cymbal and drum series names (e.g. "Zildjian A Custom", "Sabian HHX", "Pearl Reference"), not just brand names.

**Q: How many drummers are in the database?**
A: The gear search covers 60+ legendary metal drummers spanning thrash, death, black, progressive, groove, and djent subgenres.
