# MetalForge Kit Builder

> Assemble a complete metal drum kit from scratch using the brands and models favored by professional metal drummers.

**URL:** https://metalforge.io/tools/kit-builder
**Type:** Free interactive web tool

## What It Does

The MetalForge Kit Builder is a step-by-step drum kit configurator built around the gear choices of real metal drummers. Select a shell brand (Tama, Pearl, DW, Sonor, Mapex…), choose a snare, add cymbals by brand and series, and complete the kit with pedals and hardware — all guided by what 60+ legendary metal drummers actually play.

Unlike the Dream Set Builder (which starts from specific drummer setups), the Kit Builder is brand-first: you choose the manufacturer and model line, then see which pros use that combination and how their kits compare.

## Step-by-Step Flow

1. **Choose shell brand** – Browse Tama, Pearl, DW, Sonor, Mapex, Ludwig, and more
2. **Select shell series** – e.g. Tama Starclassic Maple vs Starclassic Bubinga
3. **Pick a snare** – Signature snares or standard models from the database
4. **Add cymbals** – Hi-hats, crashes, ride, chinas, splashes by brand and series
5. **Configure pedals** – Single or double-bass pedal systems
6. **Review & save** – See the complete kit with all specs and related drummer profiles

## FAQ

**Q: What is the MetalForge Kit Builder?**
A: A free brand-first drum kit configurator at metalforge.io/tools/kit-builder that guides you through selecting shells, snare, cymbals, and pedals using models verified from 60+ pro metal drummer setups.

**Q: How is the Kit Builder different from the Dream Set Builder?**
A: The Kit Builder is brand-first — you start by choosing a manufacturer and work through the model hierarchy. The Dream Set Builder is drummer-first — you start by picking a drummer's rig and remix it. Both output a complete shareable kit.

**Q: Which drum brands are available in the Kit Builder?**
A: Tama, Pearl, DW, Sonor, Mapex, Ludwig, SJC, ddrum, Orange County, and additional brands represented in the MetalForge drummer database.
